================Introduction============
Using C# and MS Access to create   small data-driven desktop apps


Introduce the basic concepts such as:
-Use C# to manage, create and connect to database
-accessing records
-Binding database fields with Objects in winforms

===========Project Details=============

-Explore Properties and Form Objects
-Working with timer, KeyPress events, Mouse Events at runtime
-Working with Designer and InitializeComponents()

C# Database objects Project:
-SQL command and SQL database objects
-Demo1
-Demo2

NorthWind Project:
-Use of C# database objects and Windows Forms
-Creating Form elements programmatically
-Single table demonstration using NorthWind database
-Connect database and find selected records from table
 and display record in datagrid view

CRUD Project:
-Multiple tables and forms
-Working with different application states for viewing, 
 updating, adding and deleting records
-Enable and disable controls according to user actions
-Passing data from one form to another
